# Sumung Waitlist Landing Page – Quick Start

This is a minimal React project for the Sumung waitlist landing.

## Quickstart

1. Unzip this project.
2. Run `npm install`
3. Run `npm start` to see the local site.

## Deploy to Netlify/Vercel

- Push to GitHub/GitLab (or connect directly).
- Go to https://netlify.com or https://vercel.com and "New Site from Git".
- Select your repo, use default build settings.
- Site is live in seconds!

## Customizing
- Update the logo in `SumungWaitlist.jsx` (currently text placeholder).
- Style: `SumungWaitlist.module.css`
- Add waitlist endpoint: wire up the form handler to POST to your Google Sheet, email, or backend.
